== INSTALL ==
1. Upload all code to Server
2. Open app/config.php and config 
3. ChMod 
         download           0777
         apps/cache         0777
         apps/backup       0777
         apps/tmp             0777
4. Create database
5. Go to http://go.use.vn/ypcore.php?registry_new_apps=true
5. Run http://domain.com/apps/backup/backup.php and enter your database user and password to install database.
6. Go to Administrator Panel by
